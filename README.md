# Caj-PDF

该项目的起因是下载的论文好多是caj格式，但caj格式的适用场景比较少，所以想转成PDF格式，但是手边能用的要么是付费的，要么是免费但每天有数量限制（个位数）。

所以经过搜索，发现了有caj2pdf的项目，但是该项目有一些依赖，而且部署起来相对麻烦，所以自己经过一番折腾，在AI的协助下，完成了该项目，从项目结构上来说，应该有些臃肿，甚至不符合开发标准，但是本人并非专业的开发人员，秉持着能用就行的态度，将该项目上传到这里。

将所有文件下载之后，运行caj2pdf_batch2.exe即可，可以一次性选择多个caj文件，并自定义输出文件夹。

双击之后，程序会检查运行所需的文件是否齐全，会在下方的窗口中显示，运行过程及运行结果均可查看，便于寻找问题原因。

希望能对需要的人有所帮助！
